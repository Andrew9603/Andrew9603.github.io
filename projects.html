狼人杀推理 Agent

查看 GitHub

把狼人杀从“模型聊天”推进到可复盘、可评测、可演化的 Agent 工作台。

NightCouncil 是一个 12 人狼人杀系统，结合确定性规则引擎、角色技能提示词、PostgreSQL 持久化、FastAPI 后端和 Vue 工作台，支持实时对局、历史回放、Benchmark、自我进化和运行观测。

系统结构

  1. engine/ 负责夜间行动、警长竞选、发言、投票、死亡结算与胜负判定
  2. app/ 编排 LLM 调用、角色技能、重试、超时和玩家视角隔离
  3. storage/ 用 PostgreSQL 保存对局、决策、Benchmark 与进化数据
  4. ui/ 提供 Lobby、Live Match、History、Benchmark、Self-evolution 等工作台页面

工程证据

  • 确定性规则引擎
  • 角色技能版本管理
  • Benchmark 排行与诊断
  • 候选技能生成与晋升流程
